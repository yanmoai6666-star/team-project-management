# 标签颜色标准化文档

## 背景介绍
统一标签颜色，提高项目管理效率

## 标签分类体系
- 问题类型
- 功能类型
- 优先级
- 状态
- 组件

## 颜色标准规则
采用GitHub标准颜色体系，按类别分配色系

## 标签列表及颜色对照表
| Label Name | Color Hex | Category | Usage |
|------------|-----------|----------|--------|
| bug | #d73a4a | 问题类型 | 软件缺陷 |
| ...（列出全部22个标签）|

## 使用指南
1. 创建Issue时根据类型选择标签
2. 优先级标签必须标注
3. 状态标签及时更新
